# ChronX Whitepaper

**Latest version:** v2.5 (March 2026)

---

## Description

The ChronX whitepaper describes the complete protocol design: cryptographic foundations (post-quantum Dilithium2 signatures), the time-lock contract architecture, tokenomics (fixed supply, 8,270,000,000 KX, no inflation), the claims resolution framework (V2 state machine, verifier staking, challenge mechanism), account recovery (social recovery with post-quantum guarantees), the secure email claim system (BLAKE3-hashed claim secrets, on-chain verification without trust), and the long-term vision for a blockchain purpose-built for multi-decade financial commitments.

## Key Topics Covered

1. **Introduction** — The problem with long-horizon financial promises in traditional and crypto finance
2. **Protocol Design** — DAG structure, PoW (SHA3-256, 20-bit difficulty), transaction types
3. **Cryptographic Foundations** — Dilithium2 (CRYSTALS-Dilithium NIST PQC), BLAKE3 hashing
4. **Time-Lock Contracts** — V0 direct-claim, V2 claims state machine, ambiguity resolution
5. **Secure Email Claims** — Claim secret protocol: BLAKE3(code) stored on-chain; recipients verify without trust
6. **Tokenomics** — Fixed supply 8,270,000,000 KX, genesis allocations, treasury release schedule
7. **Claims Resolution Architecture** — Verifier network, oracle framework, challenge/slash mechanism
8. **Account Recovery** — Post-quantum social recovery without custodians
9. **100-Year Vision** — Humanity stake, generational wealth, long-horizon governance
10. **Security Analysis** — Quantum threat model, attack surfaces, mitigation
11. **Roadmap** — Phase 1–5, from mainnet launch to DAO governance
