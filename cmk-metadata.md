## Description

ChronX is a Layer 1 blockchain protocol that enables time-locked cryptocurrency transfers with zero fees. Users can create on-chain "Promises" — funds locked until a future date, enforced by the protocol itself. No intermediary, no custodian, no counterparty risk.

The protocol supports locking funds for up to a century into the future, with post-quantum cryptographic security (CRYSTALS-Dilithium2, NIST FIPS 204).

## Key Differentiators

1. **Zero Fees Forever** — No gas, no transaction fees. The protocol is designed so the amount sent always equals the amount received.
2. **Time-Lock Promises** — Lock funds on-chain for any duration up to a century. The protocol enforces the release date — no trusted third party.
3. **Post-Quantum Security** — CRYSTALS-Dilithium2 (NIST FIPS 204) provides cryptographic security against quantum computing attacks.
4. **Email-Based Claims** — Send KX to anyone via email, even if they don't have a wallet yet. Recipients claim funds with a secure code.
5. **Fixed Supply** — Exactly 8,270,000,000 KX. No inflation, no minting, no burning.
